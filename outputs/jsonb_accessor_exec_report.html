Methodology. Produced by CI Shadow Runner: it ingests a repo's GitHub Actions history, builds a duration-weighted dependency graph, finds the removable edge, then runs the experiment on real infrastructure before proposing anything. All four runs above are public and verifiable on a throwaway fork (NWelde/jsonb_accessor). Correctness was tested on the full 186-variant matrix; the speed figure used a representative subset so both runs are green and the only variable is the one line — the saving (lint removed from the critical path) is independent of matrix size. Every number came from a run that actually happened.
